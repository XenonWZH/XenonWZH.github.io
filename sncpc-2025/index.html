ICPC 2025 陕西省赛 游记

发表于 2025-05-11 更新于 2025-05-12 分类于 日常 ， 行记 阅读次数： Disqus：

上大学以来第一场，或许也是最后一场 ICPC。或许以后会离算法竞赛越来越远吧。

Day \(x, \ x \in \mathbb{Z}_{\leq 0}\)

在水群时发现学校的 ACM 大群发了省赛报名，犹豫了一下决定报名。看到群里面有人找队友马上就去联系了。

考试前夕发现省赛和程序设计基础的期末考试撞了，为了补考还麻烦去申请了盖章的参赛证明 泥交稳定发挥。在五一时和队友训练了两三场 然后被各种唐氏问题折磨，打铁++。

Day 1

考试那天早期赶地铁，和队友到西电签到，拍了几张照就赶过去看热身赛了。然而到场的时候已经开始 30min 了 看了一眼榜决定写 A，然而想了一会儿没啥特别的思路，通读了所有题反而发现 B 更简单，在封榜的时候 A 了。

接下来是正式赛，三人分三组找签到题。先看到了 A 题和 C 题比较签，C 题观察了一会儿找出做法就是不停找 \(\gcd\)，10min 交了一发过了。这时看榜开 L，观察了一下猜答案是平均数，和队友讨论了一下确定结论，交了一发过了。

接下来看榜跟 J 题和 G 题。J 题看了一眼不太擅长就交给队友做了，G 题摸索一下找个最大值最小值，把队友从机子上赶下来快速写完过掉。甚至此时冲到了 rank3，如此美妙的开局！ 接下来就开始想 A，不难发现按照颜色分类，处理成区间计算相邻的距离即可，于是又把在写 C 的队友赶下来 进行实现，简单 debug 之后交上去过掉。

看榜开 K 题，看着像博弈论，算了一下把结果处理成分段函数，两段都是一次函数好计算极值，于是双把在写 J 的队友赶下来 进行实现，交了一发 WA 掉，把代码打印下来，边吃发的麦麦 这薯条是真难吃啊 边 debug，发现边界情况处理错了，于是叒把在写 J 的队友赶下来 改掉交上去过掉。

看榜单剩下的就是 C 题和 D 题了，看着并没有特别的思路，看队友在给 J 题 debug 就给队友造了几个 J 的样例 然后非常不幸地都是正确的。于是想了一下 CD 推了一下性质，推的时候队友找到问题就把 J 题过了。依然没有什么思路于是再读了遍题 然后发现眼瞎第一遍没读 D 的数据范围，读了之后马上确定 D 题是状压 DP，写了一会儿交上去 WA 了。此时已经基本燃尽了，也快封榜了。造了好几个样例都没发现问题。结果最后造了个小样例发现初始状态没处理好，马上改了之后在封榜时把 D 题过了。

接下来看滚榜，过题不算很多且罚时不太优，预料到大概率是银首。不出所料结果正式队伍 rank 22，银牌 rank 6，在近乎单挑的情况下算是尽力了。上台领奖拿到了第一个 ICPC 牌子。

于是接下来回学校紧锣密鼓地补数学作业……

之后看了一下 C 发现就是一道 SB 暴搜题，痛失金牌 QAQ

Day \(x, \ x \in \mathbb{Z}_{> 1}\)

现在回头看，大学最终读了数学专业，似乎和 CS 相关但现在看来本科阶段关系甚小，课内知识虽说能掌握但耗费精力巨大。在远离算法竞赛许久后回归却发现并不能跟得上 ICPC 所需的水平。在后续课内和其他的压力下，算法竞赛也只会越来越远吧。

开学后开始学习专业课，却发现数分高代学起来并不容易。一方面挣扎于数学，另一方面却也看到了同学有且不止专业课的耀眼成绩，越发觉得自己平凡。看着变幻莫测的强基政策和不确定的未来，自己想得太多，一直纠结于不少事情，又做得太少，导致一直以来精神状态挺“丧”的。

高考发挥得不错来到了一所好学校，也要开始接受自己的平凡吧。算法竞赛估计以后没时间也不太会碰了，就看以后在这所学校的六七年能活出怎样的色彩吧。
