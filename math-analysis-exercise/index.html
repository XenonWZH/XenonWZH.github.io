第一章 集合与映射

1.1 集合

T1(4)

证明以下结论：\(\{ (x, y) \in \mathbb{Z}^{2} : x^{2} - 2y^{2} = 65 \} \neq \varnothing\)。

取 \((x, y) \in \{ (x, y) \in \mathbb{Z}^{2} : x^2 - 2y^2 = 65 \}\)，则有 \(x^{2} - 2y^{2} = 65\)。故 \(x^{2} \equiv 2y^{2} \pmod 5\)。 又有 \(0^{2} \bmod 5 = 0, 1^{2} \bmod 5 = 1, 2^{2} \bmod 5 = 4, 3^{2} \bmod 5 = 4, 4^{2} \bmod 5 = 1\)，且 \(2 \cdot 0^{2} \bmod 5 = 0, 2 \cdot 1^{2} \bmod 5 = 2, 2 \cdot 2^{2} \bmod 5 = 3, 2 \cdot 3^{2} \bmod 5 = 3, 2 \cdot 4^{2} \bmod 5 = 2\)。 故当且仅当 \(5|x, 5|y\) 时满足条件。此时有 \(25 | x^{2} - 2y^{2}\)，而 \(25 \nmid 65\)，故矛盾。 故 \(\{ (x, y) \in \mathbb{Z}^{2} : x^{2} - 2y^{2} = 65 \} = \varnothing\)。

T8

设 \(X\) 是一个集合。令 \[\mathscr{R} = \{ R \subseteq X \times X : \text{对任意的} \ S \subseteq X \times X \ \text{均有} \ R \circ S = S \circ R \}.\] 证明 \(\mathscr{R} = \{ \varnothing, \Delta_{X} \}\)，其中 \(\Delta_{X} = \{ (x, x) : x \in X \}\)。

显然有 \(\varnothing \circ S = S \circ \varnothing = \varnothing, \Delta_{X} \circ S = S \circ \Delta_{X} = S\), 则 \(\varnothing, \Delta_{X} \in \mathscr{R}\)。现讨论其他元素不满足条件。 设 \(R \in \mathscr{R}\)，有下列情况。 - 若存在 \((x, y) \in R \ (x \neq y)\), 则取 \(S = \{ (y, x) \}, R' = R \setminus \{ (x, y) \}\)。 故 \(R \circ S = \{ (y, y) \} \cup R' \circ S\)，且 \(S \circ R = \{ (x, y) \} \cup S \circ R'\) 故 \((y, y) \in S \circ R'\)。而 \(S = \{ (x, y) \}\)，故矛盾； - 若存在 \((x, x) \in R, (y, y) \not\in R \ (x, y \in X)\)，则取 \(S = \{ (x, y) \}, R' = R \setminus \{ (x, x) \}\)。 故 \(R \circ S = \varnothing, S \circ R = (x, y)\)，矛盾。 综上，则 \(\mathscr{R} = \{ \varnothing, \Delta_{X} \}\)。

1.2 映射与函数

T11

设 \(f\) 与 \(g\) 均是定义在 \(\mathbb{Z}\) 上的非常值函数，且对任意的 \(x, y \in \mathbb{Z}\) 有 \[ \begin{gather*} f(x + y) = f(x)g(y) + g(x)f(y), \\ g(x + y) = g(x)g(y) - f(x)f(y). \end{gather*} \] 求 \(f(0)\) 和 \(g(0)\) 的所有可能值。

取 \(y = 0\)，有 \[ \begin{cases} f(x)(1 - g(0)) = g(x)f(0) \\ g(x)(1 - g(0)) = f(x)f(0) \end{cases} \implies \begin{cases} (1 - g(0))f(x) - f(0)g(x) = 0 \\ f(0)f(x) + (1 - g(0))g(x) = 0 \end{cases} \] 故由 Cramer 定理得，当且仅当 \[ \begin{vmatrix} 1 - g(0) & -f(0) \\ f(0) & 1 - g(0) \end{vmatrix} = (1 - g(0))^{2} + f(0)^{2} = 0 \] 时，\(f(x), g(x)\) 有非零解，即 \(f, g\) 可为非常值函数。 故 \(f(0) = 0, g(0) = 1\) 为唯一解。

T12

设 \(f\) 是定义在 \(\mathbb{R}\) 上的一个函数，证明它可写成一个奇函数与一个偶函数之和。

设 \(f(x) = g(x) + h(x)\)，其中 \(g(x), h(x)\) 分别为奇函数，偶函数。则 \(f(x) = -g(x) + h(x)\). 解得 \(g(x) = \frac{f(x) - f(-x)}{2}, h(x) = \frac{f(x) + f(-x)}{2}\)，故 \(f\) 可写成一个奇函数与一个偶函数之和。

T17(4)（三角形不等式（triangle inequality））

设 \(x \in \mathbb{R}\)，用 \(\| x \|\) 表示 \(x\) 与离它最近的整数之间的距离，即 \(\| x \| = \min\limits_{n \in \mathbb{Z}} |x - n|\)，证明：对任意的 \(x, y \in \mathbb{R}\) 有 \(\| x + y \| \leq \| x \| + \| y \|\)。

不难验证 \(\| x \|\) 以 \(1\) 为最小正周期。故不妨设 \(x, y \in \left[ -\frac{1}{2}, \frac{1}{2} \right]\)，即证 \[\min\{ |x + y|, |x + y - 1|, |x + y + 1| \} \leq |x| + |y|\] 又有 \[\min\{ |x + y|, |x + y - 1|, |x + y + 1| \} \leq |x + y| \leq |x| + |y|\] 故原命题成立。

T19

两个定义在 \(\mathbb{R}\) 上的周期函数的乘积是不是周期函数？

不是。例如 \(f(x) = \sin x, g(x) = \sin \sqrt{ 2 }x\) 均是周期函数。 而对于 \(h(x) = f(x)g(x) = \sin x \sin \sqrt{ 2 }x = -\frac{1}{2}(\cos((1 + \sqrt{ 2 })x) + \cos((1 - \sqrt{ 2 })x))\)，有 \(\frac{1 + \sqrt{ 2 }}{1 - \sqrt{ 2 }} \not\in \mathbb{Q}\)，则 \(h(x)\) 不是周期函数。

T23

设函数 \(f \colon \mathbb{Z}_{\geq 0} \to \mathbb{Z}_{\geq 0}\) 满足 \(f(n + 1) > f(f(n)) \ (\forall n \in \mathbb{Z}_{\geq 0})\)，证明：对任意的非负整数 \(n\) 有 \(f(n) = n\)。

有 \(f(n) > f(f(n - 1)) > f(f(f(n - 2))) > \cdots > f(f(\cdots f(0) \cdots))\)，则 \(f(n)\) 至少比 \(n\) 个非负整数大，即最坏情况有 \(f(n) \geq n\)。 故有 \[f(n + 1) > f(f(n)) \geq f(n)\] 即 \(f(x)\) 严格单调递增。 则由 \(f(n + 1) > f(f(n))\) 推出 \(n + 1 > f(n)\)，故 \(f(n) = n\)。

1.3 集合的运算

T9

设 \(X\) 是一个集合，现考虑 \(2^{X}\) 上的一个关系 \[R = \{ (A, B) : A, B \subseteq X \ \text{且} A \triangle B \text{是有限集} \}.\] 证明：\(R\) 是 \(2^{X}\) 上的一个等价关系。

有 \(x \triangle x = \varnothing, x \triangle y = y \triangle x\)，即关系 \(R\) 满足自反性与对称性。故只需证传递性。 设 \((A, B), (B, C) \in R \ (A, B, C \subseteq X)\)，且 \(A \triangle B = P, b \triangle C = Q\)，则 \(P, Q\) 均为有限集。则 \(A = B \triangle P, C = B \triangle Q\)。 则有 \[ \begin{align*} A \triangle C &= (B \triangle P) \triangle (B \triangle Q) \\ &= (B \triangle B) \triangle (P \triangle Q) \\ &= \varnothing \triangle (P \triangle Q) \\ &= P \triangle Q \end{align*} \] 又 \(P \triangle Q\) 为有限集，则 \(A \triangle C\) 为有限集，\((A, C) \in R\)。故满足传递性，则 \(R\) 是 \(2^{X}\) 上的一个等价关系。

T15（容斥原理）

设 \(X\) 是一个有限集，\(A_{1}, A_{2}, \cdots, A_{n}\) 均是 \(X\) 的子集。若用 \(|S|\) 表示 \(X\) 的子集 \(S\) 的元素个数，试利用关系式 \(|S| = \sum\limits_{a \in X} \chi_{S}(a)\) 证明： > \[\left| \left( \bigcup\limits_{k = 1}^{n} A_{k} \right)^{c} \right| = |X| + \sum\limits_{j=1}^{n} (-1)^{j} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} |A_{i_{1}} \cap A_{i_{2}} \cap \cdots \cap A_{i_{j}}|.\]

即证 \[ \begin{align*} \sum\limits_{a \in X} \chi_{\left( \bigcup\limits_{k = 1}^{n} A_{k} \right)^{c}}(a) &= \sum\limits_{a \in X} \chi_{X}(a) + \sum\limits_{j=1}^{n} (-1)^{j} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} \sum\limits_{a \in X} \chi_{A_{i_{1}} \cap A_{i_{2}} \cap \cdots \cap A_{i_{j}}}(a) \\ \iff \sum\limits_{a \in X} \chi_{\bigcap \limits_{k = 1}^{n} A_{k}^{c}}(a) &= \sum\limits_{a \in X} 1 + \sum\limits_{a \in X} \sum\limits_{j=1}^{n} (-1)^{j} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} \chi_{A_{i_{1}}}(a) \chi_{A_{i_{2}}}(a) \cdots \chi_{A_{i_{j}}}(a) \\ \iff \sum\limits_{a \in X} \prod\limits_{k=1}^{n} \chi_{A_{k}^{c}}(a) &= \sum\limits_{a \in X} \left( 1 + \sum\limits_{j=1}^{n} (-1)^{j} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} \chi_{A_{i_{1}}}(a) \chi_{A_{i_{2}}}(a) \cdots \chi_{A_{i_{j}}}(a) \right) \\ \iff \sum\limits_{a \in X} \prod\limits_{k=1}^{n} \left( 1 - \chi_{A_{k}}(a) \right) &= \sum\limits_{a \in X} \left( 1 + \sum\limits_{j=1}^{n} (-1)^{j} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} \chi_{A_{i_{1}}}(a) \chi_{A_{i_{2}}}(a) \cdots \chi_{A_{i_{j}}}(a) \right) \\ \iff \sum\limits_{a \in X} \prod\limits_{k=1}^{n} \left( 1 - \chi_{A_{k}}(a) \right) &= \sum\limits_{a \in X} \left( 1 + \sum\limits_{j=1}^{n} (-1)^{j} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} \chi_{A_{i_{1}}}(a) \chi_{A_{i_{2}}}(a) \cdots \chi_{A_{i_{j}}}(a) \right) \\ \end{align*} \] 故只需证 \[\prod\limits_{k=1}^{n} \left( 1 - \chi_{A_{k}}(a) \right) = 1 + \sum\limits_{j=1}^{n} (-1)^{j} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} \chi_{A_{i_{1}}}(a) \chi_{A_{i_{2}}}(a) \cdots \chi_{A_{i_{j}}}(a)\] 由排列组合知上式成立。

T17

设 \(n \geq 2\)，\(A_{1}, A_{2}, \cdots, A_{n}\) 均是有限集。用 \(|S|\) 表示集合 \(S\) 的元素个数。证明： \[|A_{1} \triangle A_{2} \triangle \cdots \triangle A_{n}| = \sum\limits_{1 \leq j \leq n} (-2)^{j - 1} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} |A_{i_{1}} \cap A_{i_{2}} \cap \cdots \cap A_{i_{j}}|.\]

设全集为 \(X\)，即证 \[ \begin{align*} \sum\limits_{a \in X} \chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{n}}(a) &= \sum\limits_{1 \leq j \leq n} (-2)^{j - 1} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} \sum\limits_{a \in X} \chi_{A_{i_{1}} \cap A_{i_{2}} \cap \cdots \cap A_{i_{j}}}(a) \\ \iff \sum\limits_{a \in X} \chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{n}}(a) &= \sum\limits_{a \in X} \sum\limits_{1 \leq j \leq n} (-2)^{j - 1} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} \chi_{A_{i_{1}} \cap A_{i_{2}} \cap \cdots \cap A_{i_{j}}}(a) \\ \end{align*} \] 故只需证 \[\chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{n}}(a) = \sum\limits_{1 \leq j \leq n} (-2)^{j - 1} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq n} \chi_{A_{i_{1}} \cap A_{i_{2}} \cap \cdots \cap A_{i_{j}}}(a)\] 使用数学归纳法证明。当 \(n = 2\) 时有 \[ \begin{align*} \chi_{A_{1} \triangle A_{2}}(a) &= |\chi_{A_{1}}(a) - \chi_{A_{2}}(a)| \\ \sum\limits_{1 \leq j \leq 2} (-2)^{j - 1} \sum\limits_{1 \leq i_{1} < \cdots < i_{j} \leq n} \chi_{A_{i_{1}} \cap \cdots \cap A_{i_{j}}}(a) &= \chi_{A_{1}}(a) + \chi_{A_{2}}(a) - 2\chi_{A_{1}}(a)\chi_{A_{2}}(a) \end{align*} \] 根据集合的特征函数取值得到上述两式相等。 假设 \(n = k\) 时成立，则 \(n = k + 1\) 时，左式有 \[ \chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{k + 1}}(a) = |\chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{k}}(a) - \chi_{A_{k + 1}}(a)| \] 右式有 \[ \begin{align*} & \sum\limits_{1 \leq j \leq k + 1} (-2)^{j - 1} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq k + 1} \chi_{A_{i_{1}} \cap A_{i_{2}} \cap \cdots \cap A_{i_{j}}}(a) \\ =& \sum\limits_{1 \leq j \leq k} (-2)^{j - 1} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq k} \chi_{A_{i_{1}} \cap A_{i_{2}} \cap \cdots \cap A_{i_{j}}}(a) + \\ & \sum\limits_{1 \leq j \leq k + 1} (-2)^{j - 1} \chi_{A_{k + 1}}(a) \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j - 1} \leq k} \chi_{A_{i_{1}}}(a) \chi_{A_{i_{2}}}(a) \cdots \chi_{A_{i_{j - 1}}}(a) \\ =& \chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{k}}(a) + \chi_{A_{k + 1}}(a) \sum\limits_{0 \leq j \leq k} (-2)^{j} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq k} \chi_{A_{i_{1}}}(a) \chi_{A_{i_{2}}}(a) \cdots \chi_{A_{i_{j}}}(a) \\ =& \chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{k}}(a) + \chi_{A_{k + 1}}(a) \left( 1 - 2\sum\limits_{1 \leq j \leq k} (-2)^{j - 1} \sum\limits_{1 \leq i_{1} < i_{2} < \cdots < i_{j} \leq k} \chi_{A_{i_{1}}}(a) \chi_{A_{i_{2}}}(a) \cdots \chi_{A_{i_{j}}}(a) \right) \\ =& \chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{k}}(a) + \chi_{A_{k + 1}}(a) \left( 1 - 2\chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{k}}(a) \right) \\ =& \chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{k}}(a) + \chi_{A_{k + 1}}(a) - 2\chi_{A_{1} \triangle A_{2} \triangle \cdots \triangle A_{k}}(a)\chi_{A_{k + 1}}(a) \end{align*} \] 根据集合的特征函数取值得到上述两式相等。于是对 \(n = k + 1\) 也成立。 综上，原式成立。
